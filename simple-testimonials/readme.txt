# Simple Testimonials Plugin

The Simple Testimonials plugin allows you to easily manage and display customer testimonials on your WordPress website.

## Installation

1. Download the plugin zip file from [GitHub](https://github.com/your-plugin-url).
2. Log in to your WordPress admin dashboard.
3. Navigate to Plugins > Add New.
4. Click on the "Upload Plugin" button.
5. Choose the downloaded zip file and click "Install Now".
6. Once installed, click on "Activate Plugin" to activate the Simple Testimonials plugin.

## Adding Testimonials

To add a new testimonial:

1. In the WordPress admin dashboard, go to Testimonials > Add New.
2. Fill in the required fields such as Customer Name, Customer Position, Company Name, Testimonial Date, and Testimonial Text.
3. Click "Publish" to save the testimonial.

## Displaying Testimonials

To display testimonials on a page or post:

1. Edit the page or post where you want to display testimonials.
2. Add the shortcode `[simple_testimonials]` to the content area.
3. Update or publish the page or post.
4. Testimonials will now be displayed in a list format wherever the shortcode is placed.

## Styling Customization

The plugin includes basic styling for testimonials, but you can customize the appearance to match your site's design by adding custom CSS.

To add custom CSS:

1. Create a file named `simple-testimonials.css`.
2. Add your custom CSS rules to this file.
3. Upload the file to the `css` folder inside the plugin directory.
4. Your custom styles will be automatically loaded and applied to the testimonials output.
